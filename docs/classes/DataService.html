DataService Class

Defined in: common/data.service.ts:11
Module: CarDetailModule

Provides exchange of data with a back-end database via the HTTP server. Three collections are implemented: cars, fillups and reminders, with CRUD methods for each. Caching is implemented for all the collections. A single method handles eventual errors.
